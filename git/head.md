## HEAD

- 현재 체크아웃된 커밋 가리킨다. (현재 작업중인 커밋)
- HEAD는 항상 작업트리의 가장 최근 커밋을 가리킨다. 작업트리에 변화를 주는 git 명령어들은 대부분 HEAD를 변경하는 것으로 시작한다.
- 일반적으로 HEAD는 브랜치의 이름을 가리키고 있다.
- HEAD를 분리한다 == 브랜치 대신 커밋에 붙이는 것을 의미한다.
